# AI Fake News Detection Using NLP

## Overview

AI-powered Fake News Detection System that classifies news articles as **Real** or **Fake** using Natural Language Processing (NLP), TF-IDF Vectorization, and a Passive Aggressive Classifier.

## Features

* Real/Fake News Classification
* NLP Text Preprocessing
* TF-IDF Vectorization
* Passive Aggressive Classifier
* Word Count
* Prediction History
* Flask-based Web Interface

## Technologies Used

* Python
* Flask
* Scikit-learn
* NLTK
* TF-IDF

## How to Run

1. Install dependencies

```bash
pip install -r requirements.txt
```

2. Run the application

```bash
python app.py
```

3. Open in browser

```text
http://127.0.0.1:5000
```
